Netflix Clone

A netflix clone web app built using ReactJS with Firebase back-end. The movie data displayed is fetched using TMBD API. The front-end is styled with Taildwind CSS. The login functionality (Firebase Authentication), user-specific data storage (Firestore) and hosting are performed using Firebase.

Lyriks

Lyriks is a Spotify-inspired music app built using ReactJS with Tailwind styling. It has a modern homepage, music player, search, lyrics, song exploration features. The state management is handled using Redux Toolkit. The data is fetched using Shazam Core API which is provided by RapidAPI. Lastly, the web app is hosted on Netlify.
