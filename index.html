Here are some of my projects

Netflix Clone

A netflix clone web app built using ReactJS with Firebase back-end. The movie data displayed is fetched using TMBD API. The front-end is styled with Taildwind CSS. The login functionality (Firebase Authentication), user-specific data storage (Firestore) and hosting are done using Firebase.

Lyriks Music Web

Lyriks is a Spotify-inspired music web built using ReactJS with Tailwind styling. It has a modern homepage, music player, search, lyrics, song exploration features. The state management is handled using Redux. The data is fetched from Shazam Core API which is provided by RapidAPI.

E-commerce

E-commerce web built using NextJS. The application includes the ability to add and edit displayed products using Sanity, cart and checkout functionalities. The integration with Stripe can allow users cover real payments (currently on test mode).

Restaurant Web

Gericht restaurant website from Figma design transformed into a fully functioning website. It has a modern look and responsive design.
